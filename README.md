# African Climate Trend Analysis – Week 0 Challenge

## Overview

This repository contains my submission for the 10 Academy Week 0 challenge focused on African Climate Trend Analysis. The project supports exploratory climate analysis for Ethiopia, Kenya, Sudan, Tanzania, and Nigeria using NASA POWER historical climate data.
